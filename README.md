# Page Capture Extension (v1.0.13)

현재 웹 페이지를 스타일과 이미지가 포함된 **단일 HTML 파일**로 캡처하여 저장하는 크롬 익스텐션입니다.

## 🚀 주요 기능
- **HTML 기반 복원:** Chrome DevTools Protocol(CDP)을 사용하여 페이지의 모든 리소스를 캡처합니다.
- **Single HTML 변환:** 데이터를 분석하여 CSS와 이미지를 인라이닝(Inlining) 처리, 단일 `.html` 파일로 변환합니다.
- **오프라인 열람:** 별도의 리소스 파일 없이 저장된 HTML 파일 하나만으로 페이지를 완벽하게 볼 수 있습니다.
- **다크 모드 UI:** 현대적이고 세련된 인터페이스를 제공합니다.

## 🛠 설치 방법 (개인용/개발자 모드)
이 익스텐션은 현재 스토어에 등록되지 않은 개발 버전이므로 아래와 같이 수동으로 설치해야 합니다.

1. 이 저장소의 코드를 다운로드하거나 클론합니다.
2. 크롬 브라우저에서 `chrome://extensions/`에 접속합니다.
3. 오른쪽 상단의 **개발자 모드**를 활성화합니다.
4. **압축해제된 확장 프로그램을 로드** 버튼을 클릭합니다.
5. 다운로드한 프로젝트 폴더를 선택합니다.

## 📖 사용 방법
1. 캡처하고 싶은 웹 페이지로 이동합니다.
2. 익스텐션 아이콘을 클릭하여 팝업을 엽니다.
3. **캡처 활성화** 토글을 켭니다.
4. **HTML 다운로드** 버튼을 누릅니다.
5. `Downloads` 폴더에서 생성된 `.html` 파일을 확인합니다.

## 📝 기술 스택
- **Manifest V3**
- **Vanilla JavaScript** (Chrome Extension API, CDP)
- **CSS3** (Custom properties, Syne/DM Mono Fonts)
